purescript-zipperarray
======================

A non-empty array with one element focused.

## Developing

### Get packages

Using NPM

```
$ npm install -g purescript spago
$ spago install
```

Using Nix Flakes

```console
$ nix develop
$ spago install
```

### Run tests

```console
$ spago --config test/test.dhall test
```
